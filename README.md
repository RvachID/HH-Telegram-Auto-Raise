# HH Telegram Auto Raise

Автоматизация поднятия резюме на hh.ru через Telegram Web с использованием Playwright.

Проект эмулирует обычные действия пользователя в Telegram Web:

* открывает чат с hh-ботом
* работает через системное меню бота
* корректно обрабатывает все сценарии (можно поднять / рано / уже поднято)

## Зачем это нужно

hh позволяет поднимать резюме не чаще одного раза в 4 часа. Через Telegram это делается кнопками, но вручную. Этот проект автоматизирует процесс без использования API hh и без обхода ограничений — только UI.

## Стек

* Node.js
* Playwright (Chromium)
* Telegram Web
* Windows Task Scheduler

## Требования

* Windows 10 / 11
* Node.js >= 18
* Установленный браузер Chromium (ставится автоматически Playwright)

## Установка

```cmd
npm install
npx playwright install
```

## Конфигурация

Настройки вынесены в файл `src/config.js`.

Основные параметры:

* username hh-бота
* тайминги ожиданий
* путь к storageState

## Запуск вручную

```cmd
node src/index.js
```

## Автоматический запуск

Для автозапуска используется Windows Task Scheduler.

Рекомендуемая схема:

* запуск каждый час
* дневное окно (08:00–22:00 МСК)
* экран выключен, без сна

Запуск осуществляется через `run.cmd`.

## Что делает скрипт

1. Открывает Telegram Web
2. Открывает меню бота
3. Переходит в «Личный кабинет»
4. Пытается поднять резюме
5. При необходимости подтверждает действие
6. Корректно завершает работу

## Безопасность

* Не используется API hh
* Нет прямых запросов
* Только действия пользователя в браузере
* Никаких обходов ограничений

## Ограничения

* Зависит от UI Telegram Web
* Требует включённый компьютер
* Не работает в режиме сна
